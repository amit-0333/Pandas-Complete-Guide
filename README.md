## 📚 Topics Covered

| # | Topic | Description |
|---|-------|-------------|
| 01 | 📌 Pandas Series | Creating, indexing, and operations on Series |
| 02 | 📊 Pandas DataFrame | Creating and exploring DataFrames |
| 03 | 🔧 DataFrame Methods | Essential methods for data exploration |
| 04 | 🔍 Boolean Indexing | Filtering data with conditions |
| 05 | 🧹 Data Cleaning | Handling missing values, duplicates, data types |
| 06 | 🔡 Vectorized String Operations | String methods using `.str` accessor |
| 07 | 🗂️ MultiIndex | Hierarchical indexing and advanced selection |
| 08 | 📅 Time Series | Date ranges, resampling, rolling windows |
| 09 | 📦 GroupBy Operations | Grouping, aggregating, and transforming data |

| 11 | 📈 Data Visualization | Plotting with Pandas + Matplotlib |
| 12 | 🌍 Case Studies | Real-world datasets and end-to-end analysis |

## 🚀 How to Run

```bash
# 1. Clone the repository
git clone https://github.com/amit-0333/Pandas-Complete-Guide.git

# 2. Navigate to the folder
cd Pandas-Complete-Guide

# 3. Install dependencies
pip install pandas numpy matplotlib jupyter

# 4. Launch Jupyter Notebook
jupyter notebook
```
